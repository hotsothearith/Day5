Task 6: Components Components in CSS are reusable blocks of code that can be used to create consistent and maintainable stylesheets. They help improve code organization, reduce redundancy, and enhance code reusability. Example: SCSS // card.scss .card { border: 1px solid #ccc; border-radius: 5px; padding: 20px; } .card-img-top { width: 100%; height: 200px; object-fit: cover; } .card-title { font-size: 1.2rem; margin-bottom: 10px; } .card-text { color: #666; }
