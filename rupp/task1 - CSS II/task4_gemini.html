Task 4: Variables, Functions, and Inheritance Variables: Variables in Sass allow you to store values like colors, fonts, and dimensions. This makes it easier to manage styles and maintain consistency throughout your project. Functions: Functions in Sass are reusable blocks of code that can take arguments and return a value. They can be used to perform calculations, create complex styles, or abstract common patterns. Inheritance: Inheritance in Sass allows you to create a base stylesheet that can be extended by other stylesheets. This promotes code reusability and helps maintain a consistent look and feel across your project. Example: SCSS // Sass code $base-font-size: 16px; $primary-color: #007bff; @mixin button($color) { background-color: $color; color: white; padding: 10px 20px; border: none; cursor: pointer; } .button { font-size: $base-font-size; @include button($primary-color); } .secondary-button { @extend .button; background-color: #ccc; } // Compiled CSS .button { font-size: 16px; background-color: #007bff; color: white; padding: 10px 20px; border: none; cursor: pointer; } .secondary-button { background-color: #ccc; }
